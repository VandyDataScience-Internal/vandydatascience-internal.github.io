The coding challenge is designed to mimic the kind of work you'd be doing as a Developer on one of our projects. Your final submission is due Monday, October 16, 2023 at 11:59pm. Due to our strict interview timeline, we will not be accepting late submissions.

To be considered for one of these 3 positions, you'll need to:

  • Upload all solutions to your personalized GitHub Classroom folder
  • Complete our final submission Google Form

The assessment consists of 3 levels. More detailed instructions are provided in the GitHub Classroom environment, which can be accessed using the links below.

Here, you can access general information about the assessment

In Level 1, you'll be asked to clean a dataset

In Level 2, you'll be asked to train a classification model using the dataset you cleaned in Level 1

In Level 3, you'll be asked to analyze and evaluate the model you built in Level 2

As a reminder, EMs, Developers, and Workshop Creators will be completing the same assessment.

Once you have uploaded your solutions to all 3 levels in GitHub Classroom, please submit your application using the Google Form below. Here, you will have the opportunity to indicate which position(s) you are applying for.
